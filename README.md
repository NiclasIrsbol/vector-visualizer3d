# QuickCards
Vector-visualizer3d - Simple vector visualizer in 3d, where a user can draw vectors, and do different vector-operations.

# How to run?
To run the application you must first install *three.js* and *vite*, using the commands below:
```
npm install --save three
npm install --save-dev vite
```
Next you run:
```
npx vite
```
And the localhost will open in browser.
